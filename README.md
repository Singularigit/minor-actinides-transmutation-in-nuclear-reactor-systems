# Transmutation – OpenMC project

## Where things go
- `model_sfr/notebooks/shared/` : baseline notebook(s)
- `model_sfr/notebooks/personal/<name>/` : each person works here
- `model_sfr/inputs/` : generated XML (ignored by git)
- `runs/` : OpenMC outputs (ignored)

## Team workflow
- Default branch: `main`
- Each person works on their own branch (`milton`, `albert`, `filip`)
- Prefer editing your personal notebook to avoid conflicts.
